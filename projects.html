LiPoker

LiPoker is an easy way to play poker online with friends. It supports videochat and messaging without the need for signup or download. Worked on a variety of fullstack tasks such as Teacher Mode, displaying player icons, and maintaining total buy-in. All code written in Python and React.

Ero Compiler

[May 2020]

Ero is a functional scripting language intended to describe and process Euclidean constructions in 2 or 3 dimensions. Wrote the tokenizer, developed classes for each geometrical figure and implemented various intersections between those objects in C++. In addition, wrote unit tests for each token being parsed by the tokenizer.
